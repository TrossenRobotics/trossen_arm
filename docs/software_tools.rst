================
Software & Tools
================

Tools and interfaces for working with Trossen Arm controllers beyond the core driver.

*   :doc:`software_tools/cli` - The ``trossen-arm`` command-line tool for interacting with controllers from the terminal without writing code.
*   :doc:`software_tools/mcp_server` - Connect the documentation, API reference, and demo scripts to your AI assistant via the Trossen Arm MCP server.
