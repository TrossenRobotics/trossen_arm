=================
Programming Guide
=================

This guide explains how to write your own applications with the Trossen Arm driver.
Start with the concepts to understand the driver's model, walk through the structure of a script, then dive into the full C++ and Python API reference.

*   :doc:`programming_guide/concepts` - The driver's model: lifecycle, modes, motion commands, frames and units, reading state, and error handling.
*   :doc:`programming_guide/writing_a_script` - The anatomy of a control script, annotated end to end.
*   :doc:`api/library_root` - The complete C++ and Python API reference.

For ready-to-run examples, see the :doc:`/getting_started/demo_scripts` catalog.
